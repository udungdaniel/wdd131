Personal Portfolio Website - Site Plan

1. Site Name

Name: CodeCraft by John Doe

This name reflects my personal brand as a developer, combining "code" (technical skill) and "craft" (creativity and polish). It is unique, professional, and clearly connected to the content of my portfolio site.

2. Site Purpose

This website is a personal portfolio to present my skills in web development. It will include completed coding projects, technical abilities, a contact form for potential clients/employers, and a blog section to share my web development journey.

3. Scenarios

  • What web development projects has John Doe completed and what technologies did he use?
  • How can I contact John Doe to collaborate or offer a job opportunity?

4. Color Schema

These colors were chosen to create a clean and modern look:

  • Primary Color (headings, navigation bar, call-to-action buttons): #0077b6
  • Accent Color (links, hover effects, highlights): #90e0ef

5. Typography

  • Headings: 'Segoe UI', sans-serif – used for all page headings and menu items for a clean, bold look.
  • Body Text: 'Verdana', sans-serif – used for paragraphs, project descriptions, and blog posts for readability.
